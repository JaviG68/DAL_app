# DAL Meeting Manager

Una aplicación para gestionar reuniones y acciones del equipo DAL.

## Despliegue

- **URL de Producción**: [https://dal-app-seven.vercel.app](https://dal-app-seven.vercel.app)
- **Repositorio**: [https://github.com/JaviG68/DAL_app](https://github.com/JaviG68/DAL_app)

## Funcionalidades

- Gestión de reuniones.
- Lista de acciones con prioridad y estado.
- Panel de seguimiento por equipo.
- Reportes detallados.
- Sincronización con archivos locales mediante File System Access API.

## Cómo usar localmente

1. Clona el repositorio.
2. Abre `index.html` en un servidor local (o simplemente arrástralo al navegador si no usas funcionalidades de red, aunque se recomienda un servidor local para APIs de navegador).
3. Conecta una carpeta local para guardar tus datos.
